401 /// Emits the IR computing the multi-sized tiling specification with two tile
402 /// sizes not exceeding `targetSize`, each divisible by `sizeDivisor`, such
403 /// that there exist numbers of tiles with these sizes that fully cover the
404 /// given iteration space `dimension` of the structured `op`.
405 ///
406 /// The computation is as follows:
407 ///
408 /// b = originalTripCount floordiv sizeDivisor
409 /// t = (targetSize + sizeDivisor - 1) floordiv sizeDivisor
410 /// d = (b + t - 1) floordiv t
411 /// s = (b floordiv d) * sizeDivisor
412 /// v = b % d
413 /// u = d - v
414 ///
415 /// where the tile sizes are `s` and `s` + `sizeDivisor`, and the numbers of
416 /// the corresponding tiles are `u` and `v`, respectively. Alternatively,
417 ///
418 /// s * u + (s + sizeDivisor) * v == original size,
419 /// where s mod sizeDivisor = 0.
420 ///
421 /// Expects all values to be positive. In some cases with the target tile size
422 /// sufficiently close to the dimension shape and non-unit divisor, it is
423 /// impossible to compute such sizes. If `emitAssertion` is set, also emit the
424 /// assertion that size computation succeeded.
425 ///
426 /// Returns the specification consisting of both tile values and the number of
427 /// tiles of each size.

498 /// All indices returned by IndexOp should be invariant with respect to
499 /// tiling. Therefore, if an operation is tiled, we have to transform the
500 /// indices accordingly, i.e. offset them by the values of the corresponding
501 /// induction variables that are captured implicitly in the body of the op.
502 ///
503 /// Example. `linalg.generic` before tiling:
504 ///
505 /// #id_2d = (i, j) -> (i, j)
506 /// #pointwise_2d_trait = {
507 /// indexing_maps = [#id_2d, #id_2d],
508 /// iterator_types = ["parallel", "parallel"]
509 /// }
510 /// linalg.generic #pointwise_2d_trait %operand, %result {
511 /// ^bb0(%operand_in: f32, %result_in: f32):
512 /// %i = linalg.index 0 : index
513 /// %j = linalg.index 1 : index
514 /// <some operations that use %i, %j>
515 /// }: memref<50x100xf32>, memref<50x100xf32>
516 ///
517 /// After tiling pass with tiles sizes 10 and 25:
518 ///
519 /// #strided = (i, j)[s0, s1, s2] -> (i * s1 + s0 + j * s2)
520 ///
521 /// %c1 = arith.constant 1 : index
522 /// %c0 = arith.constant 0 : index
523 /// %c25 = arith.constant 25 : index
524 /// %c10 = arith.constant 10 : index
525 /// operand_dim_0 = dim %operand, 0 : memref<50x100xf32>
526 /// operand_dim_1 = dim %operand, 1 : memref<50x100xf32>
