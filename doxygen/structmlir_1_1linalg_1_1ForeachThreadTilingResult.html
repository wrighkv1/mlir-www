Detailed Description

Rewrite a TilingInterface op to a tiled scf.foreach_thread, applying tiling by numThreads.

If non-empty, the mapping is added as an attribute to the resulting scf.foreach_thread. Zero tile sizes indicate that the dimension is not tiled, and can be thought of as tiling by the full size of data. It is the user's responsibility to ensure that numThreads is a valid tiling specification (i.e. that only tiles parallel dimensions, e.g. in the Linalg case).
